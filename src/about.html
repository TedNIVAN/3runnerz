About

How it works?

3Runnerz is a blockchain-based running system based on Hedera Hashgraph.
It allows runners to register for marathons and rewards them with RUNz tokens and NFTs depending on their performances.
3Runnerz ecosystem is made of 2 applications: 1. The 3Runnerz portal where runners can register 2. The 3Runnerz Go mobile application that runners will need to install in order to participate in the marathons. Applications Overview 1. 3Runnerz portal 2. 3Runnerz Go Runners register on the app using their Header AccountId.
